Access and Develop on the Brain

Users can remotely access and start developing on the brain via SSH access. The only requirement is to be connected to the same network of your Amiga.

Sign up for a farm-ng-user accountâ€‹

1. Request SSH access to our Fleet Customer Support via email (coming up soon)â€‹

2. You will receive a confirmation email with a link for sign-up along with instructionsâ€‹

At this time, we only support Google accounts. While not required, we highly encourage you to sign up using your institutional email. If your institution does not use Google accounts, you may be able to create a Google account using your existing institutional email address.

3. On our fleet manager website, click on loginâ€‹

4. Select a Google account and proceed to the next pageâ€‹

5. Follow the instructions to create a valid usernameâ€‹

Please note that once set, this username cannot be changed.

6. Click on "Set Name"â€‹

7. Now you will need to register an SSH keyâ€‹

An SSH key is like a secret handshake that lets your computer talk safely to another computer without using a password.

8. Open a terminal window on your computer and typeâ€‹

cd ~ # This will take you to your home directory
ssh-keygen # This creates an ssh key
TIP

You can skip this step if you already have a ~/.ssh/id_rsa.pub public key on your computer that you'd like to use.

9. When prompted, press Enter twice to use a blank passphraseâ€‹

10. Use the following command to display your public SSH keyâ€‹

cat ~/.ssh/id_rsa.pub # Copy it with Shift + Ctrl + C

11. Copy your SSH key to the fleet manager website (New SSH Key)â€‹

12. Assign a name to that key (e.g., my-lenovo-pc)â€‹

13. Click on Add SSH Keyâ€‹

14. Email our Fleet Customer Support to inform that you have signed up successfullyâ€‹

They will grant you access to your robot.

Congratulations, you're done setting up your account!

INFO

Each SSH key corresponds to a single PC. If you need to access the Amiga from multiple PCs, you will need to repeat steps 7-13 for each PC.

(Recommended) Configure your SSH Connectionsâ€‹

Now that you have an SSH key and your account is ready, let's configure your SSH connections.

This can be done via the terminal or VS Code.

  • Terminal
  • VS Code

Open a terminal window and type the following command:

gedit ~/.ssh/config # This will open your .ssh/config
# Here we're using gedit as our text manager, feel free to choose another one (e.g., vi, nano)

Hit Enter and then add the following lines to your file:

Host <robot-name>
  HostName <ip-address>
  User farm-ng-user-<username>

Replace <robot-name> with your robot name (e.g., element-vegetable) and <ip-address> the robot's local IP address.

Replace <username> with your username (e.g., john-doe). Make sure to save the file before you close it!

TIP

You can download Fing to scan all devices connected to your network and figure out your robot's IP address.

1. Click on the "Open a Remote Window" (blue button on the bottom left corner of your VS Code screen)â€‹

2. Click on "Connect to Host"â€‹

3. Click on "Configure SSH Hosts"â€‹

4. Click on the first option that pops up (/home/user/.ssh/config)â€‹

5. Add the following lines to your fileâ€‹

Host <robot-name>
  HostName <ip-address>
  User farm-ng-user-<username>
  # Replace <robot-name> with your robot name (e.g., element-vegetable).
  # You should receive your robot name via email
  # Replace <username> with your username (e.g., john-doe)
  # and <ip-address> with the robot's local IP address.
TIP

You can download Fing to scan all devices connected to your network and figure out your robot's IP address.

6. Save and close the documentâ€‹

Accessing the brainâ€‹

You can access the brain via a terminal or VS Code.

  • Terminal
  • VS Code

Connecting to your robot via the terminal is simple. Open a terminal window and type:

ssh farm-ng-user-<your-farm-ng-username>@<ip-address>
# e.g., farm-ng-user-john-doe@10.0.4.116
TIP

If you have configured your SSH Connections (.ssh/config) you can connect to the robot by simply typing:

ssh <robot-name>
# Make sure to replace <robot-name> with your actual robot name

You should now be connected to your robot!

1. Click on the "Open a Remote Window" (blue button on the bottom left corner of your VS Code)â€‹

2. Select the hostâ€‹

3. Hit Enter if asked to continueâ€‹

Congratulations, you are connected to your robot via SSH!
